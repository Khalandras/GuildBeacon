# GuildBeacon v0.2.0

Premiere version jouable pour officiers de guilde.

## Added

- Tableau de bord `/gb dashboard` (candidats, inbox, balise).
- Panneau reglages `/gb config`.
- Balise : rotation messages/canaux, jitter, anti-spam, poster maintenant.
- Inbox : MP + chat guilde (mots-cles).
- Pipeline candidats : statuts, enrichissement Raider.IO, export JSON.
- FR + EN.

## Commands

- `/gb dashboard` - tableau officiers
- `/gb beacon start|stop|preview|now`
- `/gb export` - export candidats (JSON)
